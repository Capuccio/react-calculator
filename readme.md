# React Calculator

Clone the project to use it locally:

```sh
git clone https://github.com/Capuccio/react-calculator.git
cd react-calculator
npm run dev
```

| Script | Action |
| ------ | ------ |
| **npm run dev** | Initialize the development server |
| **npm run build** | Create the *dist* folder for production |
